# 功能简介

1 基础功能：
* 用户进入商品推荐页面，商品列表页推荐秒杀商品
* 进入商品详情页可获取商品详情，普通商品（非秒杀类商品）可直接购买；秒杀类商品需要在秒杀开始后购买
* 下单页面可进行支付，支付成功后根据本次购买结果推荐商品

2 拓展功能：
* 用户验证登录，对VIP用户展示不同界面以及商品价格等

# 功能简介

## 1 基础功能
* 用户获取短信验证码注册，利用账号名称和账号密码验证登录
* 用户进入商品推荐页面，商品列表页推荐秒杀商品
* 商品界面会显示未开始的秒杀活动（显示开始倒计时）和正在进行的秒杀活动
* 进入商品详情页可获取商品详情，普通商品（非秒杀类商品）可直接购买；秒杀类商品需要在秒杀开始后购买，用户下单享受秒杀价格，但存在数量限购
* 下单页面可进行支付，支付成功后刷新页面可即时看到商品销量的增加和库存的减少
